Selamat datang di repository resmi **SNISHOP.ID** - Landing Page Utama untuk Ekosistem Platform Digital SNISHOP NETWORK INDONESIA.

## 🌟 Tentang SNISHOP
SNISHOP adalah penyedia solusi platform digital komprehensif yang dirancang untuk membantu mahasiswa, profesional, dan bisnis UMKM di Indonesia bertransformasi ke arah digital dengan mudah dan terjangkau. 

Landing Page ini berfungsi sebagai "Gerbang Utama" (Gateway Hub) yang menghubungkan pengguna ke berbagai sub-layanan dan aplikasi spesifik di dalam ekosistem SNISHOP.

## 🛠️ Tech Stack
Website landing page ini dibangun dengan fokus pada performa maksimal, SEO yang kuat, dan animasi fluid tanpa membebani browser:
- **HTML5 & CSS3** murni (Vanilla, tanpa framework berat).
- **JavaScript Modern (ES6)** untuk animasi interaktif (Infinite Scroll Slideshow, Parallax, Cursor Glow).
- **Responsive Web Design** (Mobile-first CSS Breakpoints).
- **Vercel** untuk Global CDN & Fast Hosting Deployment.

## 🚀 Fitur Unggulan UI/UX
- **Smooth Infinite Looping Slideshow**: Sistem carousel murni menggunakan JS Cloning + CSS Translate untuk performa 60FPS.
- **Micro-Interactions**: Hover magnetic buttons, tilt layout 3D pada card pricing, dan subtle glow effects.
- **True Responsive**: Tampilan dijamin sempurna dari perangkat mobile terkecil (320px) hingga layar Ultra-Wide Desktop.
- **Dynamic Blogs**: Sistem blog statis yang di-generate via JavaScript builder script untuk Technical SEO Optimization.

## ⚙️ Cara Menjalankan Secara Lokal
Karena ini murni static files, cara menjalankannya sangat mudah:
1. Clone repository ini: 
   ```bash
   git clone https://github.com/SIRAJcrypto11/SNISHOP-NETWORK-INDONESIA.git
   ```
2. Buka folder proyek di Text Editor (VSCode / Cursor):
   ```bash
   cd SNISHOP-NETWORK-INDONESIA
   ```
3. Gunakan eksistensi seperti **Live Server** di VSCode, lalu klik "Go Live" pada `index.html`.

## 🔄 Deployment (Otomatis ke Vercel)
Website ini sudah terhubung dengan **Vercel** untuk Continuous Deployment. Setiap kali ada perubahan kode yang di-*push* ke branch `main`, Vercel akan secara otomatis membangun (re-build) dan memperbarui website live dalam hitungan detik.

```bash
# Tambah perubahan
git add .
git commit -m "Update layout & add ConverHub app"

# Push ke GitHub (Vercel otomatis update)
git push origin main
```
